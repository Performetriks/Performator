# Performator
Performator is a small, fully open-source performance testing framework that can execute anything you can write in Java!

It comes with a powerful and easy to extend reporting engine, that can report the collected metrics out-of-the-box as HTML, CSV, JSON and to databases.
Created by performance engineers for performance engineers, it is rich in features that help creating the tests and collecting the metrics you need for getting the job done properly. 

Because it is fully code based, your tests can be stored in a version control system and integrated and executed in a pipeline of your favorite CI/CD tool. This allows other project team members to run the performance tests on demand, without the need to have access to the code.
